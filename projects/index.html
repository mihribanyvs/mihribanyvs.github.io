ProtSpark

Designed a pipeline to analyze protein sequences coming from Protein Data Bank (PDB) using PySpark. Checked the cosine similarities between the sequences using embeddings from a pertained model (ProtBert). The dataset was about 10 GB. We used 3 Virtual Machines with CPU’s, which ran the whole program in about 20 minutes. For benchmarking we varied the number of cores, the size of the memory used and the number of workers.

Ising model on GPU

Simulated the Ising model using a Jetson Nano. Implemented two different ways to do the parallelization on the device and benchmarked the results.

Transformer From Scratch for Protein Data

Designed a transformer to predict the folding structure of proteins using the sequence information embedded using the pertained ProtBert model. The model designed was aimed to understand the mechanics behind AlphaFold. Using a custom loss function we achieved not so high accuracies but we were able to reproduce most of the tertiary structures in addition to secondary structures.

Ultrametric Patterns

Created ultra metric patterns and showed the storage capacity in a hop field network using these patterns.

Türkiye Earthquake Data Analysis

Worked on the earthquake measurement in Türkiye in the last 100 years. Implemented the Gutenberg-Richter Law to predict the future earthquakes. Analyzed the data as a time-series data. The results showed that the dataset is not a time-series and cannot be predicted as such.

Information Bottleneck Theory in Deep Neural Networks

Created multiple networks for different datasets and trained them with changing the network parameters (activation functions, optimizers, loss functions). Estimate the mutual information between the input and the layers, between the layers and the labels to observe the learning in the information plane.
